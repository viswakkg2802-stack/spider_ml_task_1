# Base Task

This folder contains the core task assets for the Fashion-MNIST model implementation.

- `notebooks/` — contains the notebook used for data loading, model training, evaluation, plotting, and output generation.
- `saved_models/` — stores the saved model weights produced by the notebook.
- `submission.csv` — generated output file containing test predictions.

The notebook in `base_task/notebooks/` is configured to save:

- model weights to `base_task/saved_models/fashion_mnist_nn_model.pkl`
- predictions to `base_task/submission.csv`
